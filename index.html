Pengolahan Citra Digital — Filtering Lab (Mask Processing)

Aplikasi single-file untuk menerapkan berbagai filter (smoothing, sharpening, edge detection, artistic) berbasis kernel secara manual pixel-by-pixel menggunakan HTML5 Canvas dan JavaScript (tanpa library eksternal).

Editable Kernel Grid Isi nilai (boleh negatif / desimal)

Boundary handling: edge replication (pixel di luar batas mengambil pixel terdekat).
Inti algoritma: convolution (mask processing) menghitung tiap pixel output dengan menjumlahkan hasil perkalian nilai tetangga dengan koefisien kernel.
